## 주요 기능

- 🚀 **Tauri 기반**: Rust 백엔드와 React 프론트엔드
- 🔒 **안전한 실행**: 샌드박스 환경에서 JavaScript 실행
- 📝 **실행 기록**: 코드 실행 이력 관리
- 💾 **코드 저장/로드**: 자주 사용하는 코드 저장
- 🎨 **모던 UI**: 직관적이고 반응형 사용자 인터페이스
- ⚡ **빠른 성능**: 네이티브 성능과 웹 기술의 결합

## 기술 스택

- **Frontend**: React 18, TypeScript, Vite
- **Backend**: Rust, Tauri 2.0
- **Testing**: Vitest, Testing Library
- **Linting**: oxlint, Prettier
- **Documentation**: RSPress

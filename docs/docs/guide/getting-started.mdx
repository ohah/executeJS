# 시작하기

ExecuteJS를 사용하여 JavaScript 코드를 안전하게 실행하는 방법을 알아보세요.

## 설치

### 요구사항

- Node.js 22 LTS (`.nvmrc` 파일 참조)
- pnpm 10+
- Rust 1.70+
- Tauri CLI

### 개발 환경 설정

1. **저장소 클론**

   ```bash
   git clone https://github.com/ohah/executeJS
   cd executeJS
   ```

2. **의존성 설치**

   ```bash
   pnpm install
   ```

3. **개발 서버 실행**
   ```bash
   pnpm tauri:dev
   ```

## 기본 사용법

### JavaScript 코드 실행

1. 메인 화면의 코드 입력 영역에 JavaScript 코드를 작성합니다.
2. "실행" 버튼을 클릭하여 코드를 실행합니다.
3. 실행 결과가 하단의 결과 영역에 표시됩니다.

### 코드 저장 및 로드

- **저장**: 현재 작성한 코드를 로컬에 저장할 수 있습니다.
- **로드**: 이전에 저장한 코드를 불러올 수 있습니다.

```
└── ...
```

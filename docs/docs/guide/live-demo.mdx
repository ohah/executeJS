# Live Demo

ExecuteJS의 라이브 데모를 통해 JavaScript 실행 기능을 직접 체험해보세요.

## 기본 사용법

다음은 ExecuteJS의 기본 기능을 시뮬레이션하는 예제입니다:

```tsx live
export default function BasicExample() {
  const greeting = 'Hello, ExecuteJS!';
  const sum = 10 + 20;
  const jsResult = 'JavaScript 실행 시뮬레이션: console.log("Hello from ExecuteJS!");';
  
  return (
    <div style={{ padding: '20px', fontFamily: 'monospace' }}>
      <h3>ExecuteJS 기본 데모</h3>
      <p><strong>인사:</strong> {greeting}</p>
      <p><strong>덧셈:</strong> 10 + 20 = {sum}</p>
      <p><strong>JavaScript 실행:</strong> {jsResult}</p>
    </div>
  );
}
```

## JavaScript 실행 시뮬레이션

실제 JavaScript 코드 실행을 시뮬레이션하는 예제입니다:

```tsx live
import { useState } from 'react';

export default function JsExecutionDemo() {
  const [code, setCode] = useState('console.log("Hello, ExecuteJS!");');
  const [result, setResult] = useState('');

  const handleExecute = () => {
    // JavaScript 실행 시뮬레이션
    const mockResult = `실행 결과: ${code.length}자 코드가 실행되었습니다.\n실제 실행: ${code}`;
    setResult(mockResult);
  };

  return (
    <div style={{ padding: '20px' }}>
      <h3>JavaScript 실행 데모</h3>
      <div style={{ marginBottom: '10px' }}>
        <textarea
          value={code}
          onChange={(e) => setCode(e.target.value)}
          style={{
            width: '100%',
            height: '100px',
            fontFamily: 'monospace',
            padding: '10px',
            border: '1px solid #ccc',
            borderRadius: '4px'
          }}
        />
      </div>
      <button
        onClick={handleExecute}
        style={{
          padding: '8px 16px',
          backgroundColor: '#667eea',
          color: 'white',
          border: 'none',
          borderRadius: '4px',
          cursor: 'pointer',
          marginBottom: '10px'
        }}
      >
        실행
      </button>
      {result && (
        <div style={{
          backgroundColor: '#f8f9fa',
          padding: '10px',
          borderRadius: '4px',
          fontFamily: 'monospace',
          whiteSpace: 'pre-wrap'
        }}>
          <strong>결과:</strong><br />
          {result}
        </div>
      )}
    </div>
  );
}
```

## 수학 연산 예제

JavaScript의 기본 수학 연산을 사용한 예제입니다:

```tsx live
export default function MathExample() {
  const numbers = [1, 2, 3, 4, 5];
  const doubled = numbers.map(n => n + n); // JavaScript의 기본 덧셈 연산
  
  return (
    <div style={{ padding: '20px' }}>
      <h3>수학 연산 예제</h3>
      <p><strong>원본 배열:</strong> [{numbers.join(', ')}]</p>
      <p><strong>2배 배열:</strong> [{doubled.join(', ')}]</p>
      <p><strong>합계:</strong> {doubled.reduce((acc, curr) => acc + curr, 0)}</p>
    </div>
  );
}
```

## 인터랙티브 데모

사용자가 직접 값을 입력하고 결과를 확인할 수 있는 인터랙티브 데모입니다:

```tsx live
import { useState } from 'react';

export default function InteractiveDemo() {
  const [a, setA] = useState(10);
  const [b, setB] = useState(20);
  const [jsCode, setJsCode] = useState('Math.random() * 100');

  const sum = a + b;
  const jsResult = eval(jsCode).toFixed(2);

  return (
    <div style={{ padding: '20px' }}>
      <h3>인터랙티브 데모</h3>
      
      <div style={{ marginBottom: '20px' }}>
        <h4>덧셈 계산기</h4>
        <div style={{ display: 'flex', gap: '10px', alignItems: 'center', marginBottom: '10px' }}>
          <input
            type="number"
            value={a}
            onChange={(e) => setA(Number(e.target.value))}
            style={{ width: '80px', padding: '5px' }}
          />
          <span>+</span>
          <input
            type="number"
            value={b}
            onChange={(e) => setB(Number(e.target.value))}
            style={{ width: '80px', padding: '5px' }}
          />
          <span>=</span>
          <strong style={{ fontSize: '18px', color: '#667eea' }}>{sum}</strong>
        </div>
      </div>

      <div>
        <h4>JavaScript 실행</h4>
        <div style={{ marginBottom: '10px' }}>
          <input
            type="text"
            value={jsCode}
            onChange={(e) => setJsCode(e.target.value)}
            style={{ width: '100%', padding: '5px', fontFamily: 'monospace' }}
            placeholder="JavaScript 표현식을 입력하세요"
          />
        </div>
        <div style={{
          backgroundColor: '#f8f9fa',
          padding: '10px',
          borderRadius: '4px',
          fontFamily: 'monospace'
        }}>
          <strong>결과:</strong> {jsResult}
        </div>
      </div>
    </div>
  );
}
```

이러한 라이브 데모를 통해 ExecuteJS의 기능을 직접 체험하고 테스트해볼 수 있습니다.
